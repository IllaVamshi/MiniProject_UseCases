# Smart Expense Tracker

A simple Python application that helps users record and analyze daily expenses.

## What it does:

- Lets you add daily expenses with date, category, amount, and description
- Stores data in a JSON file
- Shows all recorded expenses
- Generates monthly summary (total + average spending)
- Provides category-wise analysis
- Identifies highest spending category
- Displays a pie chart for expense distribution

## Usage:

Just run it in IDLE or terminal:

```bash
python expense_tracker.py
```

## Notes:

- The file `expenses.json` will be created automatically when you add your first expense.
- Make sure you have matplotlib installed for charts.

## Requirements:

- Python 3
- matplotlib

Install matplotlib using:
```bash
pip install matplotlib
```
